Etiquetas Estructurales

Dan forma y semántica al esqueleto de una página web.

<html> Raíz

Definición

Es el elemento raíz de todo documento HTML. Envuelve todo el contenido.

Sintaxis

<html lang="es">
  ...contenido...
</html>

Buenas prácticas

  • Siempre incluir el atributo lang para accesibilidad.
  • Debe contener solo <head> y <body>.
<head> Metadatos

Definición

Contiene los metadatos: título, estilos y scripts. No se muestra en pantalla.

Sintaxis

<head>
  <meta charset="UTF-8">
  <title>Mi página</title>
  <link rel="stylesheet" href="css.css">
</head>

Buenas prácticas

  • Incluir siempre charset, viewport y title.
  • Cargar CSS antes de cerrar el <head>.
<body> Contenido

Definición

Contiene todo el contenido visible de la página.

Sintaxis

<body>
  <h1>Hola mundo</h1>
  <p>Bienvenido al blog.</p>
</body>

Buenas prácticas

  • Solo puede haber un <body> por documento.
  • Cargar scripts antes de cerrar </body>.
<header> <main> <footer> Semántica

Definición

Etiquetas semánticas que delimitan las zonas principales del sitio.

Sintaxis

<header>🌸 Mi Blog</header>
<main>
  <p>Contenido principal</p>
</main>
<footer>© 2025</footer>

Demo visual

MAIN — contenido
FOOTER

Buenas prácticas

  • Mejoran accesibilidad y SEO.
  • Reemplaza <div id="header">.

Etiquetas de Texto

Formatean y estructuran el contenido textual del documento.

<h1> — <h6> Encabezados

📖 Definición

Definen títulos y subtítulos. h1 es el más importante; h6 el menos.

💻 Sintaxis

<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Sección</h3>

🎨 Demo visual

H1 — Título grande

H2 — Subtítulo

H3 — Sección pequeña

✅ Buenas prácticas

  • Solo un <h1> por página para SEO.
  • No saltar niveles de jerarquía.
<p> Párrafo

📖 Definición

Define un párrafo de texto con márgenes automáticos.

💻 Sintaxis

<p>Este es un párrafo de ejemplo.</p>

✅ Buenas prácticas

  • No anidar párrafos dentro de otros párrafos.
  • Usar para bloques de texto continuo.
<strong> & <em> Énfasis

📖 Definición

<strong> indica importancia (negrita). <em> indica énfasis (cursiva).

💻 Sintaxis

<p>Es <strong>muy importante</strong>
estudiar <em>todos los días</em>.</p>

🎨 Demo visual

Es muy importante estudiar todos los días.

✅ Buenas prácticas

  • Preferir strong/em sobre b/i por semántica.
<a> Enlace

📖 Definición

Crea hipervínculos a otras páginas, archivos o secciones.

💻 Sintaxis

<a href="https://developer.mozilla.org"
   target="_blank">
  Visita MDN 🌐
</a>

🎨 Demo visual

Visita MDN 🌐

✅ Buenas prácticas

  • Usar texto descriptivo, no "haz clic aquí".
  • Agregar rel="noopener" con target="_blank".
